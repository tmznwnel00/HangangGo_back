Species Classification API Demo

This demo uses the AI for Earth Species Classification API, which identifies over 5,000 plant and animal species. Visit the API home page for more information or to request an API key. The code used to train our model is available on GitHub.

The API runs on the AI for Earth API backend, which makes it easy for anyone working on machine learning for sustainability and environmental science to turn their models into APIs and Web-based applications. If you’re working in this space and you need cloud computing resources to bring your work to life, check out our AI for Earth grants program.
